# DocuGen Documentation

DocuGen is a tool to generate Zeal/Dash docsets from online documentation. It supports various documentation formats and can handle JavaScript-heavy sites using Playwright or Qt.

## Features
- Fetches documentation from URLs.
- Discovers and follows links within the same documentation scope.
- Rewrites assets (CSS, images) to work locally.
- Parses symbols (classes, functions, etc.) for various formats (Sphinx, Docusaurus, Rustdoc).
- Generates a SQLite index for Zeal/Dash.
- GUI and CLI interfaces.
